# Demo Script

## Opening

Mantle Alpha Sentinel is an autonomous AI alpha analyst for Mantle. It watches smart-money cohorts and protocol flow, explains emerging alpha, and commits its predictions on-chain.

## Demo Steps

1. Open the dashboard.
2. Click `Run Alpha Scan`.
3. Show the agent timeline:
   - observe smart wallets
   - score protocol anomalies
   - generate signal
   - prepare on-chain commitment
4. Show the Data Source panel:
   - fixture snapshot for repeatable alpha detection
   - real Mantle RPC chain id / latest block probe
5. Open the top signal.
6. Explain the evidence:
   - wallets involved
   - protocol or asset
   - anomaly score
   - expected horizon
7. Show the signal hash.
8. Open the commitment preview and show the `recordSignal` calldata for Mantle Sepolia.
9. Show the agent track record:
   - hit rate
   - hits / misses / pending
   - outcome resolver source snapshot
10. Open the Mantle Sepolia commitment transaction:
    - https://sepolia.mantlescan.xyz/tx/0xf7d6c4b51eb42bafdbf9b79f7f630c621750346557cc37a597c0a63863ba5c51
11. Close with the core benchmark claim: the agent commits before outcomes are known, then gets scored later.

## Judge-Friendly Framing

This is not a wallet risk scanner. It is a public alpha agent with a verifiable prediction trail.

The key claim is simple: if the agent publishes alpha, it should be accountable for whether that alpha worked.
